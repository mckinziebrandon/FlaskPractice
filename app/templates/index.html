Flask Practice

Me learning various aspects of the Flask Microframework and tinkering around with with JavaScript/JQuery. General focus is interacting with databases and generating clean data visualizations. I've also started making small JavaScript games because it's fun. Feel free to have a look around while I figure out what needs to be done.

Overview

Here are some of the main features/packages used so far (detailed overview coming soon):

  • Database: Flask-SQLAlchemy.
  • REST API: Flask-RESTful.
  • Modern JS: The Games are written using TypeScript transpiled to the modern JS ES2015 syntax.
  • Reference Material: I've started compiling a list of notes as I learn new tools/frameworks in the reference dropdown above. Currently it contains info/notes from Eloquent JavaScript (E-Book), JQuery, animation, and, of course, Flask.

One of the main goals for the site is to easily see the source code alongside the functionality. A great example is Bootswatch, the site that provides the API I use for the theme selector on the top right of the site.
